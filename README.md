# Book Tracker

## Автор
Артём Половинкин

## Описание
Приложение для управления списком прочитанных книг с возможностью фильтрации и сохранения данных.

## Инструкция по запуску
1. Установите Python и необходимые библиотеки.
2. Запустите скрипт с помощью команды `python book_tracker.py`.

## Примеры использования
1. Добавьте данные о книге.
2. Используйте фильтры для поиска книг по жанру или количеству страниц.
